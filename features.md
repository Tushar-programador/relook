# 🚀 FeedSpace - Feature Specification & Pricing Breakdown

## 📌 Overview

FeedSpace is a **UGC (User Generated Content) + Feedback SaaS platform** designed to help businesses collect, manage, and showcase customer feedback in the form of:

- 🎥 Video testimonials  
- 🎤 Audio feedback  
- 💬 Text messages  

This document defines all features categorized by subscription plans along with their **usage, purpose, and real-world use cases**.

---

# 🧩 CORE FEATURE CATEGORIES

1. Authentication & User Management  
2. Project / Brand Management  
3. Feedback Collection System  
4. Media Upload System  
5. Dashboard & Moderation  
6. Embed Widget System  
7. Analytics  
8. AI Features  
9. Billing & Subscription  

---

# 🟢 FREE PLAN

## 🎯 Purpose
- User acquisition  
- Allow small users to experience core value  
- Drive upgrades  

## 👤 Target Users
- Instagram sellers  
- Freelancers  
- Early-stage startups  

---

## 🔐 1. Authentication & User Management

### Features
- Email/password signup & login  
- JWT authentication  

### Usage
User creates an account and logs into dashboard  

### Use Case
A freelancer signs up to collect testimonials from clients  

---

## 🏷️ 2. Project / Brand Management

### Features
- Create 1 project  
- Unique public slug  

### Usage
Each project represents a brand  

### Use Case
User creates:

yourapp.com/feedback/my-brand


---

## 📥 3. Feedback Collection

### Features
- Collect:
  - Text  
  - Audio  
  - Video  
- Max 50 responses/month  
- Max video length: 30 sec  

### Usage
User shares link with customers  

### Use Case
Instagram seller adds link in bio → collects reviews  

---

## 📤 4. Media Upload

### Features
- Upload audio/video  
- Basic compression  

### Usage
Customer records video testimonial  

### Purpose
Capture authentic feedback  

---

## 📊 5. Dashboard

### Features
- View feedback  
- Approve / Reject  

### Usage
User filters and manages submissions  

### Use Case
Approve only good testimonials  

---

## 🌐 6. Embed Widget

### Features
- Basic iframe embed  
- Shows approved feedback  
- FeedSpace branding included  

### Usage
Add testimonials to website  

### Use Case
Startup embeds testimonials on landing page  

---

## ⚠️ Limitations
- FeedSpace branding visible  
- No analytics  
- Limited storage  
- No advanced customization
